# Changelog

## [1.0.0] - 2026-02-02

### Added - Lucidia Extensions
- Multi-zone system with 5 unique zones (Plaza, Library, Dev Lab, Comm Tower, Innovation Park)
- NATS event bus integration for BlackRoad OS
- Zone Manager with automatic zone switching based on activity
- Activity trigger system (git, test, deploy, search, file, terminal)
- Real-time stats and agent tracking
- Graceful degradation (works without NATS)
- Debug mode for detailed logging

### Changed
- Package name: `pixelhq` → `lucidia-workspace`
- Extended README with Lucidia documentation
- Updated package.json metadata

### Maintained from PixelHQ
- Original MIT license
- File watching and monitoring
- WebSocket server functionality
- iOS Pixel Office app compatibility
- Bonjour auto-discovery
- Secure pairing system
